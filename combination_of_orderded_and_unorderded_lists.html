Most Common Text Formatting Tags in HTML

  1. strong

    • Use Case: Indicates strong importance or emphasis.
    • Best Practice:
      • Use it to highlight important words or phrases.
      • Avoid using it for stylistic purposes alone; use CSS for styling.

  2. <em>

    • Use Case: Indicates emphasis.
    • Best Practice:
      • Use it to emphasize certain words or phrases.
      • Avoid using it solely for visual effects; use CSS for styling.

  3. <u>

    • Use Case: Underlines text.
    • Best Practice:
      • Use it sparingly since underlining is often associated with hyperlinks.
      • Prefer CSS for text decoration.

  4. <del>

    • Use Case: Represents deleted or removed text.
    • Best Practice:
      • Use it in conjunction with the <ins> tag to represent edits or changes in a document.
      • Consider using CSS to style the deleted text.

  5. <sup>

    • Use Case: Formats text as superscript (raised above the baseline).
    • Best Practice:
      • Use it to indicate footnotes, mathematical exponents, or other references.
      • Consider styling with CSS for consistency.

  6. <sub>

    • Use Case: Formats text as subscript (lowered below the baseline).
    • Best Practice:
      • Use it for chemical formulas, footnotes, or other related notations.
      • Consider styling with CSS for consistency.
