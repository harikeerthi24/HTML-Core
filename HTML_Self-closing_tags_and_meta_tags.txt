- Meta tags:
       - Meta tags are HTML tags that provide information about the web page. 
       - They are placed in the head section of the HTML document and are not visible on the page itself. 
       - They provide metadata about the HTML document such as author, description, keywords, and other information that can be used by search engines to index the page. 
       - You can use meta tags to specify the character set used by the document, refresh rate, and more.
